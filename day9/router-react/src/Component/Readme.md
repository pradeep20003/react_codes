Axios :-

Axios is js library for making api request.[GET,POST  ,DELETE , PUT]

GET :- data get 
POST :- Data insert /save 
DELETE :- data delete 
PUT :- update data 

Fetch and Axios difference :- 

1. by using this fetch we can only fetch the data.
1. Axios we can get , post , delete 

2. In fetch error handling complex.
2. In Axios error handling easy.

3. convert the data mannually -> data -> json
3. Automatically converted into the json form.
